## 📝 第2步：创建内容类型 (15-20分钟)

### 2.1 进入内容类型构建器

1. 登录管理面板
2. 点击左侧菜单 **Content-Type Builder**

### 2.2 创建 Essays (随笔) - 8个字段

点击 **Create new collection type**

**Display name**: `Essay`

按顺序添加以下字段：

#### 字段 1: Category (分类)
- **Field type**: `Enumeration`
- **Name**: `category`
- **Values** (每行一个):
  ```
  ESSAY
  THOUGHTS
  LIFESTYLE
  ```
- **Default value**: `ESSAY`

#### 字段 2: Title (标题)
- **Field type**: `Text`
- **Name**: `title`
- **Required field**: ✅ 勾选

#### 字段 3: Excerpt (摘要)
- **Field type**: `Text`
- **Name**: `excerpt`

#### 字段 4: Content (完整内容)
- **Field type**: `Rich Text`
- **Name**: `content`

#### 字段 5: Date (发布日期)
- **Field type**: `Date`
- **Name**: `date`
- **Default value**: 点击 `Today` 按钮

#### 字段 6: Featured Image (封面图)
- **Field type**: `Media`
- **Name**: `featuredImage`
- **Media type**: `Single media`
- **Allowed types**: 只勾选 `Images`

#### 字段 7: Slug (URL 标识)
- **Field type**: `UID`
- **Name**: `slug`
- **Attached to field**: 选择 `title`

#### 字段 8: Published (是否发布)
- **Field type**: `Boolean`
- **Name**: `published`
- **Default value**: 勾选 ✅ (True)

点击 **Finish** → **Save**

### 2.3 创建 Tutorials (教程) - 10个字段

点击 **Create new collection type**

**Display name**: `Tutorial`

按顺序添加字段：

#### 字段 1-2: 基本信息
1. **Title** - Text (Required)
2. **Description** - Text

#### 字段 3-5: 分类信息
3. **Level** - Enumeration
   - Values: `初级` `中高级` `全阶段` `高阶`
   - Default: `全阶段`

4. **Status** - Enumeration
   - Values: `更新中` `已完结`
   - Default: `更新中`

5. **Chapters** - Number (Integer)
   - Default: `1`

#### 字段 6-8: 内容字段
6. **Icon** - Enumeration
   - Values: `code` `layers` `zap`

7. **Content** - Rich Text

8. **Cover Image** - Media (Single, Images only)

#### 字段 9-10: 元数据
9. **Slug** - UID (Attached to: title)
10. **Published** - Boolean (Default: True)

点击 **Finish** → **Save**

### 2.4 创建 Tools (工具集) - 6个字段

**Display name**: `Tool`

字段列表：
1. **Title** - Text (Required)
2. **Description** - Text
3. **Icon** - Enumeration (`barChart` `droplet` `fileText` `search`)
4. **URL** - Text
5. **Cover Image** - Media (Single, Images)
6. **Slug** - UID (Attached to: title)

点击 **Finish** → **Save**

### 2.5 创建 Comments (评论) - 7个字段 + 关系

**Display name**: `Comment`

#### 基本字段 (1-5):
1. **Author Name** - Text (Required)
2. **Content** - Text (Required)
3. **Email** - Email (可选)
4. **Rating** - Number (Integer)
   - Minimum: `1`
   - Maximum: `5`

5. **Approved** - Boolean (Default: False)

#### 关系字段 (6-7):
6. **Essay** - Relation
   - Type: `One-to-many`
   - Target: `Essay`
   - Description: An essay has many comments

7. **Tutorial** - Relation
   - Type: `One-to-many`
   - Target: `Tutorial`
   - Description: A tutorial has many comments

**注意**: 评论只能关联到随笔或教程之一

点击 **Finish** → **Save**

### 2.6 验证内容类型

检查是否成功创建了4个内容类型：

- [ ] Essay (8个字段)
- [ ] Tutorial (10个字段)
- [ ] Tool (6个字段)
- [ ] Comment (7个字段 + 2个关系)

### 5.3 创建 API 服务

创建文件: `src/services/api.ts`

```typescript
import axios from 'axios';

const API_URL = 'http://localhost:1337/api';

// 随笔类型
export interface ApiEssay {
  id: number;
  attributes: {
    category: 'ESSAY' | 'THOUGHTS' | 'LIFESTYLE';
    title: string;
    excerpt: string;
    date: string;
    slug: string;
    published: boolean;
  };
}

// 教程类型
export interface ApiTutorial {
  id: number;
  attributes: {
    title: string;
    description: string;
    level: string;
    status: string;
    chapters: number;
    icon: 'code' | 'layers' | 'zap';
    slug: string;
    published: boolean;
  };
}

// 工具类型
export interface ApiTool {
  id: number;
  attributes: {
    title: string;
    description: string;
    icon: 'barChart' | 'droplet' | 'fileText' | 'search';
    url: string;
    slug: string;
  };
}

class ApiService {
  // 随笔
  async getEssays(): Promise<ApiEssay[]> {
    const response = await axios.get(`${API_URL}/essays`, {
      params: { 'filters[published][$eq]': true, sort: 'date:desc' }
    });
    return response.data.data || [];
  }

  // 教程
  async getTutorials(): Promise<ApiTutorial[]> {
    const response = await axios.get(`${API_URL}/tutorials`, {
      params: { 'filters[published][$eq]': true, sort: 'createdAt:desc' }
    });
    return response.data.data || [];
  }

  // 工具
  async getTools(): Promise<ApiTool[]> {
    const response = await axios.get(`${API_URL}/tools`);
    return response.data.data || [];
  }
}

export const api = new ApiService();
export { ApiEssay, ApiTutorial, ApiTool };
```

### 5.4 创建数据转换工具

创建文件: `src/utils/transformData.ts`

```typescript
import type { ApiEssay, ApiTutorial, ApiTool } from '../services/api';
import type { Essay, Tutorial, Tool } from '../types';

export function transformEssays(apiEssays: ApiEssay[]): Essay[] {
  return apiEssays.map(essay => ({
    id: essay.id.toString(),
    category: essay.attributes.category,
    date: formatDate(essay.attributes.date),
    title: essay.attributes.title,
    excerpt: essay.attributes.excerpt,
    link: `/essays/${essay.attributes.slug}`,
  }));
}

export function transformTutorials(apiTutorials: ApiTutorial[]): Tutorial[] {
  return apiTutorials.map(tutorial => ({
    id: tutorial.id.toString(),
    icon: tutorial.attributes.icon,
    title: tutorial.attributes.title,
    description: tutorial.attributes.description,
    level: tutorial.attributes.level,
    status: `${tutorial.attributes.status} (${tutorial.attributes.chapters} 章)`,
    link: `/tutorials/${tutorial.attributes.slug}`,
  }));
}

export function transformTools(apiTools: ApiTool[]): Tool[] {
  return apiTools.map(tool => ({
    id: tool.id.toString(),
    icon: tool.attributes.icon,
    title: tool.attributes.title,
    description: tool.attributes.description,
    link: tool.attributes.url,
  }));
}

function formatDate(dateString: string): string {
  const date = new Date(dateString);
  return date.toLocaleDateString('zh-CN', { year: 'numeric', month: '2-digit', day: '2-digit' });
}
```

import { useEffect, useState } from 'react';
import { ScrollReveal } from '../ScrollReveal';
import type { Essay } from '../../types';
import { api } from '../../services/api';

function EssayCard({ essay }: { essay: Essay }) {
  return (
    <>
      <div className={styles.essayMeta}>
        <span>{essay.category}</span>

## 🆘 遇到问题?

### 问题 1: 无法创建 Strapi 项目

**症状**: 卡在认证提示

**解决**:
- 手动按 ↓ 键选择 Skip
- 或关闭当前进程，重新运行命令

### 问题 2: 前端无法获取数据 (CORS)

**症状**: 控制台显示 `CORS policy` 错误

**解决**:
1. 在后端 `config/middlewares.js` 中添加:
   ```javascript
   {
     name: 'strapi::cors',
     config: {
       origin: ['http://localhost:5173'],
       credentials: true,
     },
   }
   ```
2. 重启后端服务器

### 问题 3: 403 Forbidden 错误

**症状**: 访问 API 返回 403

**解决**:
1. 检查 **Public** 角色权限
2. 确保勾选了 **find** 和 **findOne**
3. 重启 Strapi

### 问题 4: 数据为空

**症状**: 页面空白，无错误

**解决**:
1. 检查是否创建了示例数据
2. 检查数据的 `published` 字段
3. 在浏览器直接访问 API 端点测试

### 问题 5: 前端编译错误

**症状**: 终端显示 TypeScript 错误

**解决**:
1. 检查 import 路径是否正确
2. 检查类型定义是否匹配
3. 清除 node_modules 重新安装:
   ```bash
   rm -rf node_modules
   pnpm install
   ```
