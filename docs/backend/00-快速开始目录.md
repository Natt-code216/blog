## 🚀 常用命令速查

### 后端操作

```bash
# 初始化项目
npx create-strapi-app@latest backend --quickstart

# 启动开发服务器
cd backend
npm run develop

# 访问管理面板
http://localhost:1337/admin

# 测试 API
curl http://localhost:1337/api/essays
```

### 前端操作

```bash
# 安装依赖
cd E:\Claude_code\blog
pnpm add axios

# 启动开发服务器
pnpm dev

# 访问前端
http://localhost:5173
```

## 🆘 常见问题

### Q1: 文档太多，不知道从哪开始？

**答**: 直接阅读 **[01-完整搭建指南](./01-完整搭建指南.md)**，它包含了所有步骤。

### Q2: 某个步骤卡住了怎么办？

**答**: 
1. 先查看该步骤的"遇到问题"部分
2. 然后查阅对应的专项文档
3. 最后查看故障排查章节

### Q3: 代码复制后报错？

**答**: 
1. 检查 import 路径
2. 确认依赖已安装 (axios)
3. 查看控制台错误信息
4. 参考 [FRONTEND_INTEGRATION.md](./FRONTEND_INTEGRATION.md) 的完整示例
