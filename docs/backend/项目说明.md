# 项目说明

## 📁 项目结构

```
blog/
├── 📄 README.md                   # 项目说明文档
├── 📄 BACKEND_SETUP.md             # 后端初始化和架构指南
├── 📄 CONTENT_TYPE_SETUP.md        # 内容类型详细配置指南
├── 📄 PERMISSIONS_SETUP.md         # 权限和媒体配置指南
├── 📄 FRONTEND_INTEGRATION.md      # 前端集成完整指南
├── 📄 INTEGRATION_SUMMARY.md       # 快速开始和文档索引 ✨
├── 📄 blog.html                    # 原始 HTML (保留)
├── 📁 src/                         # React + TypeScript 源代码
│   ├── 📁 components/              # 组件
│   │   ├── Navbar/                 # 导航栏
│   │   ├── Hero/                   # 首页 Hero
│   │   ├── Essays/                 # 随笔板块
│   │   ├── Tutorials/              # 教程板块
│   │   ├── Tools/                  # 工具集板块
│   │   ├── Footer/                 # 页脚
│   │   └── ScrollReveal/           # 滚动动画包装组件
│   ├── 📁 hooks/                   # 自定义 Hooks
│   │   ├── useScrollSpy.ts         # 导航高亮
│   │   └── useIntersectionObserver.ts  # 滚动揭示动画
│   ├── 📁 types/                   # TypeScript 类型定义
│   ├── 📁 services/                # API 服务 (待创建)
│   └── 📁 utils/                   # 工具函数 (待创建)
└── 📁 backend/                     # Strapi 后端 (待创建)
```
